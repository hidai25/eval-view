# EvalView Examples

Real-world examples of testing popular AI agent frameworks with EvalView.

## Quick Start

1. Choose an example below
2. Follow setup instructions in each folder
3. Run: `evalview run --pattern examples/<folder>/test-case.yaml`

## Examples

| Framework | Use Case | Folder |
|-----------|----------|--------|
| LangGraph | Research agent with multi-step reasoning | [langgraph/](langgraph/) |
| CrewAI | Multi-agent content creation team | [crewai/](crewai/) |
| RAG Support | Customer support with knowledge base | [rag-support/](rag-support/) |
| Dev Assistant | Code generation and analysis | [dev-assistant/](dev-assistant/) |
| OpenAI Assistants | General-purpose assistant | [openai-assistants/](openai-assistants/) |
